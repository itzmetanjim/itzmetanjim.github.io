## 🛠️ Technology

- Simple HTML/CSS website
- Vite for development and building
- GitHub Pages for hosting

## 🚀 Development

To run locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

To deploy:

```bash
npm run deploy
```
